Export user emails to .csv file for use in address book or mailing list

== Description ==

A plugin that exports user email address.

You can even export the emails by role and registration date range.

= Features =

* Exports all emails fields
* Exports emails by metadata
* Exports emails by role
* Exports emails by date range

For feature request and bug reports, please use my support form (http://josiahdwilliams.com/support/).

== Installation ==

For an automatic installation through WordPress:

1. Go to the 'Add New' plugins screen in your WordPress admin area
1. Search for 'Export Email to CSV'
1. Click 'Install Now' and activate the plugin
1. Go the 'Users' menu, under 'Email to CSV'



For a manual installation via FTP:

1. Upload the `jw-export-email-to-csv` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' screen in your WordPress admin area
1. Go the 'Users' menu, under 'Email to CSV'


To upload the plugin through WordPress, instead of FTP:

1. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
1. Go the 'Users' menu, under 'Email to CSV'

== Frequently Asked Questions ==

= How to use? =

Click on the 'email to CSV' link in the 'Users' menu, choose the role and the date range or don't select anything if you want to export all users, then click 'Export'. That's all!
